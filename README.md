# Ethosoft-Haber-Analizi
Ethosoft bünyesinde geliştirdiğim bu yeni proje sayesinde haberlerin özetlerine dünya gerilimine katkısına Kıyamet Saati projesi üzerinden tahminlerle yapay zeka arama motoruyla ve haberin taraflı olup olmadığını tehit eden yapay zeka aracıyla tek uygulamada yapılabilmektedir
Çalıştırmak için gerekli kütüphaneleri terminalden pip install komutu ile kurmanız ve calistir.bat'a çift tıklamanız yeterlidir
